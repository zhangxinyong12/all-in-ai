### 常见问题

**Q1: 需要学Python吗？**
推荐学习。虽然可以用Node.js，但Python是AI生态的主流语言，学习成本不高（1-2周入门）。

**Q2: 需要深入数学吗？**
应用开发不需要高深数学。了解基础概念即可：向量、矩阵、概率统计。

**Q3: 大专学历会被卡吗？**
应用开发岗位更看重实战能力。作品集和项目经验比学历更重要。
